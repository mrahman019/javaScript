*WHAT IS JAVA SCRIPT?

JavaScript is a High level, Interpreted,
 multi-pradigm programming language.

 It runs mainly in the browser but also on server (Node.Js)

 It's dynamic (types can change), eventdriven, and supports Object-oriented, functional, and imperative styles.

 Often paired with HTML + CSS to build webapps.
